# RWKV-7 1.5B × 我们的三值 CIM 芯片

`config [EXTERNAL]` · `sizing [ESTIMATE]` · `chip [TARGET]`

**判决：1GB 单片 ✅（甜点区正中，最稳的边缘主力档）**

## 参数拆解
- C = 2048，L = 24（RWKV-7 公开配置 `[EXTERNAL]`）
- ~1.5B 参数；Uncheatable Eval 同尺寸打赢 Qwen2.5-1.5B（7.969 < 8.124，`[EXTERNAL]`）

## 三值后内存 `[ESTIMATE]`（见 [`../sizing.py`](../sizing.py)）
| 项 | 大小 |
|---|---|
| 三值权重（transformer，1.58 bit） | 239 MB |
| emb + head（int8，不可三值化的大词表税） | 268 MB |
| 递归状态（int8，batch=1） | ~2 MB |
| **片上合计** | **~0.51 GB** |

## 上不上片
- 1GB `[TARGET]`：✅ 宽裕，单片
- 3GB `[TARGET]`：✅
- 单片 / 多片：**单片**

## 适用场景
AI 眼镜 / 可穿戴 / 录音笔上的**常开私有 agent**：语音命令、转写、翻译，以及实时同传的候选档位。够聪明、又能塞进亚瓦封套——这是 1GB 这颗片最自然的归宿。
