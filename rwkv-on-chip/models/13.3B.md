# RWKV-7 13.3B × 我们的三值 CIM 芯片

*设计点参数为圆整估算；内存为算术估算；芯片容量是设计目标，尚未流片。*

**判决：1GB ❌ · 3GB 临界 ✅(~2.96GB 擦边) / 否则 2 片模组**

## 参数拆解
- C = 4096，L ≈ 61（⚠️ 该档具体配置**未独立核实**；L 取到约对上 13.3B 标称值。请以 RWKV 官方配置为准）
- ~13B 参数；Uncheatable Eval 打赢 Qwen2.5-14B（6.843 < 6.951）

## 三值后内存（见 [`../sizing.py`](../sizing.py)）
| 项 | 大小 |
|---|---|
| 三值权重（transformer，1.58 bit） | 2425 MB |
| emb + head（int8，不可三值化） | 537 MB |
| 递归状态（int8，batch=1） | ~16 MB |
| **片上合计** | **~2.96 GB** |

## 上不上片
- **1GB 片**：❌
- **3GB 片**：✅ **但临界**（~2.96GB，几乎贴满）
- 单片 / 多片：3GB 片擦边单片，**更稳的做法是 2 片拼模组**——RWKV 逐层递归、状态恒定，按层切天然好切

## 适用场景
最强的离线私有 reasoning（小型私有服务器 / 高端工作站）。到这个规模，单片不再是甜点——它更适合"几片拼一个私有模组"的形态。把它放进 zoo，是为了把**容量阶梯的上沿**也标清楚：哪里单片、哪里该拼片。
