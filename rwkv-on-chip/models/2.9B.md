# RWKV-7 2.9B × 我们的三值 CIM 芯片

*配置为 RWKV 官方公开参数；内存为算术估算；芯片容量是设计目标，尚未流片。*

**判决：1GB 单片 ✅（甜点区上沿——1GB 能吃下的最强档）**

## 参数拆解
- C = 2560，L = 32（RWKV-7 公开配置）
- ~2.9B 参数；Uncheatable Eval 同尺寸打赢 Qwen2.5-3B（7.486 < 7.722）

## 三值后内存（见 [`../sizing.py`](../sizing.py)）
| 项 | 大小 |
|---|---|
| 三值权重（transformer，1.58 bit） | 497 MB |
| emb + head（int8，不可三值化） | 336 MB |
| 递归状态（int8，batch=1） | ~3 MB |
| **片上合计** | **~0.83 GB** |

## 上不上片
- **1GB 片**：✅ 单片（留~17% 余量）
- **3GB 片**：✅
- 单片 / 多片：**单片**

## 适用场景
对话质量、端侧 coding agent、更强离线推理——需要"够聪明"而非"够省"时的 1GB 上限档。再往上（4B+）就压过 1GB 线了，得上 3GB 目标片。

---
*数字由 `sizing.py` 生成（算术估算）；四闸近无损为小规模 PoC 趋势；芯片为综合+仿真+设计阶段，未流片。*
