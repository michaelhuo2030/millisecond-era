# RWKV-7 7.2B × 我们的三值 CIM 芯片

*配置为 RWKV 官方公开参数；内存为算术估算；芯片容量是设计目标，尚未流片。*

**判决：1GB ❌ · 3GB 单片 ✅（3GB 目标片的主力档）**

## 参数拆解
- C = 4096，L = 32（RWKV-7 公开配置，7B 档 = L32/C4096）
- ~7B 参数

## 三值后内存（见 [`../sizing.py`](../sizing.py)）
| 项 | 大小 |
|---|---|
| 三值权重（transformer，1.58 bit） | 1272 MB |
| emb + head（int8，不可三值化） | 537 MB |
| 递归状态（int8，batch=1） | ~8 MB |
| **片上合计** | **~1.81 GB** |

## 上不上片
- **1GB 片**：❌（光三值权重就 1.27GB，超）
- **3GB 片**：✅ 单片（留~40% 余量）
- 单片 / 多片：**3GB 目标片单片**；1GB 片需 2 片拼模组

## 适用场景
桌面 / 边缘工作站级的私有助手，更强 reasoning。这一档是"把密度做到 3GB"之后单片就能吃下的甜点上沿——不需要堆先进制程，靠三值 + 无 ADC 在成熟节点上拿够用的密度。
