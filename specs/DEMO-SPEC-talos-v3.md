# DEMO 规格 · "TALOS-V3" —— 三值 transformer + HDC，在便宜 FPGA 上真跑起来（2026-05-30）

> Michael 拍板：fork TALOS-V2（Karpathy microGPT 的 RTL，DE1-SoC 实测 ~53k tok/s）→ 升级成**更大一点的小模型 + 三值权重 + 接 HDC**，**把速度真跑起来**。演示主打"快"。
> 这是 我们的软件 demo的**硬件孪生**：软件那边给客户看软件 demo，硬件这边给人看"一个真 transformer 在 ¥2000 板子上飞、只用加减法"。

## 一句话目标（演示的"哇"时刻）
**"一个真的会写字的 transformer，在一块二手 ¥2000 的板子上，只用加减法（没有乘法器）跑到每秒几千~几万个字；旁边再当场灌噪声——三值还对，INT8 全乱。"**

## 两条路（同一块板，讲两个故事）
- **A · 主线 = 小模型 + 全塞片上 + 飞快**（showcase，主打速度）。模型整个住在片上 BRAM → 不搬数据 → 几千~几万 tok/s。
- **B · 副线 = 大模型 + 放 DDR + 慢**（对比，主打"也塞得下"）。把一个 8B 三值（1.6GB）塞进板载 DDR，流式跑，~几十 tok/s，证明"真大模型也能装进来，只是要等 ASIC 才快"。

## 选板（先便宜起步，想 flex 再上大的）
| 阶段 | 板子 | 片上能塞的三值模型 | 演什么 |
|---|---|---|---|
| **起步（推荐）** | **Kria KV260**（¥1500–3000） | ~1000 万参数全塞片上 | 一个"小但能写出像样文字"的字符级 GPT，跑到几千~几万 tok/s |
| 拔高 flex | **Alveo U200/U280**（~¥5000） | ~1.5 亿全塞片上 = **GPT-2-small（124M）** | 一个**叫得出名字**的真模型全片上飞 |
| 副线 B | 同板（KV260 的 4GB DDR） | 8B 三值放 DDR | "大模型也塞得下"，慢演示 |

## 要建什么（在 TALOS-V2 基础上改，4 步）
1. **Fork TALOS-V2 的 RTL**（transformer 数据通路现成）。
2. **把 MAC 换成三值**：权重 {−1,0,+1} → 乘法变成"加 / 减 / 跳过零"，**砍掉所有乘法器**。量 LUT/功耗 vs 原版 INT8/定点。
3. **接一个 HDC 引擎**：bind=XOR、bundle=popcount/majority、cleanup=Hamming（全是位操作，FPGA 原生）。演一个确定性的小 trace 或 item-memory 召回（每次跑结果一模一样）。
4. **噪声注入 demo**（最炸的一幕）：往三值计算里灌扰动 → 输出还对；同样灌进 INT8 版 → 输出崩。把 E7 那个发现搬到真硬件上当场演。

## 要量什么（= 拿得出手的证据，4 个数）
1. **速度**：三值、全片上的 tok/s（对标 TALOS-V2 的 53k）。
2. **省**：三值 MAC vs INT8 MAC 在同板上的 LUT/功耗（证明"只用加减法"的便宜）。
3. **稳**：三值 vs INT8 在注入噪声下的正确率（E7 现场版）。
4. **HDC**：bind/cleanup 的延迟 + 确定性（重复跑 bit 级一致）。

## 顺序（增量，先打通再加料）
0. **【前提/Gate-0】先把 TALOS-V2 → bitstream 真烧进板子跑通**（现在测试通过但从没端到端综合过——这是 elephant #1）。工具齐：aux-mac 上的 `vivado-vm`（Zynq UltraScale+ 直接支持）、EBAZ 已能开机。
1. 三值化 MAC → 量 tok/s + LUT vs 原版。
2. 主线 A：把模型撑到片上上限，测速度。
3. 加 HDC 引擎。
4. 噪声注入 demo。
5. 副线 B：8B 三值放 DDR，流式跑（慢演示）。
6. 产出：**板子上跑的视频 + 一篇创始人内容**（比"¥150 跑 Linux"那篇更硬核）。

## 诚实边界（别把它说成芯片）
- FPGA 的速度 = **证明设计对 + 三值省多少**，**不是产品速度**（ASIC 慢 10–50 倍这件事反过来：真芯片更快）。
- FPGA **没有 ReRAM**，演不出"模拟省电"；省电/密度故事留给 ASIC。
- 主线 A 的"快"靠"全塞片上"；一旦模型超过片上、走 DDR，就回到副线 B 的慢区间。三值的作用是把"塞得进"的天花板顶高 ~10 倍。
